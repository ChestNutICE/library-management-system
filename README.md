# 图书管理系统

这是一个使用 Python 3.12、PySide6 和 SQLite 开发的 Windows 单机图书管理系统。

## 已实现功能

- 管理员和读者登录、密码修改及角色权限隔离。
- 图书新增、查询、修改、停用和库存管理。
- 读者新增、查询、修改和停用。
- 借书、还书、30 天借期、最多同时借 5 本、重复借阅拦截。
- 逾期识别、逾期费用计算、个人及全局借阅记录。
- 实时统计首页、热门图书排行和操作日志。
- SQLite 数据库备份、校验和恢复。
- 图书分类维护、分类筛选、图书和读者重新启用。
- 借阅续借一次、到期提醒及读者联系电话汇总。
- Excel 批量导入导出和统计 PDF 报表。
- 图书封面持久化保存，支持 USB 条码枪/二维码枪扫描 ISBN 查询。
- 管理员子账号及图书、读者、借阅、报表四类细粒度权限。
- 每日首次启动自动创建数据库备份。

## 源码运行

在项目根目录打开 PowerShell：

```powershell
.\.venv312\Scripts\python.exe main.py
```

初始管理员账号：`admin`  
初始管理员密码：`admin123`

首次登录后建议立即在“账户设置”中修改密码。

## 自动化测试

```powershell
.\.venv312\Scripts\python.exe -m pytest -q
```

## EXE 运行

进入 `dist\LibraryManagementSystem`，双击 `LibraryManagementSystem.exe`。程序会在 EXE 同级创建 `data` 和 `backups` 目录，请不要只复制 EXE 文件，应复制整个文件夹。

## 基本使用顺序

1. 管理员登录。
2. 在“图书管理”中新增图书。
3. 在“读者管理”中新增读者。
4. 在“借阅管理”中选择读者和图书办理借书。
5. 选择未归还记录办理还书。
6. 在“首页”“逾期记录”和“操作日志”中查看运行情况。
7. 在“数据维护”中定期创建数据库备份。
8. 使用“图书分类”维护分类；使用“数据维护”导入导出 Excel 或 PDF。
9. 超市式 USB 条码枪或二维码枪连接后，将光标放入图书搜索框直接扫描。

## Excel 导入格式

- 图书表头：`ISBN、书名、作者、出版社、总库存、馆藏位置`
- 读者表头：`用户名、姓名、电话`
- 导入读者的初始密码统一为 `123456`，应提醒读者首次登录后修改。

## 数据文件

- 源码运行：`data\library.db`
- 打包运行：`dist\LibraryManagementSystem\data\library.db`
- 默认备份：运行目录下的 `backups` 文件夹

## 常见问题

- 登录失败：检查用户名、密码以及账户是否停用。
- 无法借书：检查库存、读者状态、是否已借同一本书或是否达到 5 本上限。
- 恢复备份后数据未刷新：关闭并重新启动程序。
- 杀毒软件拦截 EXE：PyInstaller 生成的未签名课程作业程序可能触发误报，可改用源码运行并保留构建记录。
